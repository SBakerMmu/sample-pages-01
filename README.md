# sample-pages-01

A sample static website hosted with [GitHub Pages](https://pages.github.com/).

## Live site

Once GitHub Pages is enabled for this repository the site is available at:

```
https://sbakermmu.github.io/sample-pages-01/
```

## What's included

| File | Purpose |
|------|---------|
| `index.html` | Main landing page |
| `styles.css` | Page styles |

## Enabling GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under *Source*, select the `main` branch and click **Save**.
3. Your site will be live within a minute.
