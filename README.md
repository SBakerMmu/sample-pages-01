# Sample-Pages-01

A sample static website hosted with [GitHub Pages](https://pages.github.com/).

## Live site

Once GitHub Pages is enabled for this repository the site is available at:
[https://sbakermmu.github.io/sample-pages-01/](https://sbakermmu.github.io/sample-pages-01/)

Uses a  Github workflow `deploystatic.yml` for deploying static content from the website folder to GitHub Pages
